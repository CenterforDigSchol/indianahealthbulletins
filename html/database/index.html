This database contains mortality data extracted from tables published in the Monthly Bulletin of the Indiana State Board of Health. Queries produce a table showing the number of deaths from the specified causes by month and geographical region. As the Monthly Bulletin only reported deaths from "important causes", data is not available for all causes of death. This mortality data does not include stillbirths. Monthly statistics are currently available for the October 1899 - December 1939 time frame. Population was based on the latest census, or is the estimated population.
